# ASSEMBLY INSTRUCTIONS

## Step 1: Print Specifications

| Item | Specification |
|------|---------------|
| Paper | White, letter size (8.5" x 11") |
| Print Quality | High quality, color where applicable |
| Single/Double Sided | Single-sided recommended for clarity |
| Font | Clear, readable (articles printed from web) |

## Step 2: Organization by Criterion

1. **Sort all materials by criterion letter** (A, B, C, E, G, S)
2. **Within each criterion**, organize by exhibit number
3. **For Criterion C**, organize by tier (Tier 1 first, then Tier 2, then Tier 3)
4. Keep forms and brief at the front of Binder 1

## Step 3: Exhibit Label Creation

Create labels for each exhibit using this format:

```
┌─────────────────────────────────────┐
│          EXHIBIT [A-1]              │
│                                     │
│  Lou Groza Award 2023 Semifinalist  │
│                                     │
│  Source: Oklahoma State Athletics   │
│  Date: November 9, 2023             │
│  Pages: 1-3                         │
└─────────────────────────────────────┘
```

**Label Placement:** Top right corner of first page of each exhibit

## Step 4: Highlighting Application

### Highlighting Principles

1. **Use yellow highlighter** for key information
2. **Do not over-highlight** - focus on essential facts
3. **Highlight consistently** across similar documents
4. **Key items to highlight:**
   - Beneficiary's name (every occurrence in headlines/key sections)
   - Award names and recognition levels
   - Statistics and rankings
   - Dates of achievements
   - Quotes about the beneficiary

### Exhibit-Specific Highlighting Instructions

#### Criterion A Exhibits:
- Highlight "Alex Hale" in headlines
- Highlight "semifinalist," "first team," "selection"
- Highlight specific statistics (27 field goals, 2.22 per game, etc.)
- Highlight "only kicker" or similar unique distinctions

#### Criterion B Exhibits:
- Highlight "International Pathway Player"
- Highlight selection criteria language
- Highlight selectivity numbers (8-10 players, top 20, etc.)

#### Criterion C Exhibits:
- Highlight article titles containing Hale's name
- Highlight publication names (Sports Illustrated, ESPN, etc.)
- Highlight author names and publication dates
- Highlight key quotes about Hale

#### Criterion E Exhibits:
- Highlight "#1," "FBS leader," ranking language
- Highlight record achievements
- Highlight expert assessments of significance

#### Criterion G Exhibits:
- Highlight "starting kicker," "starter"
- Highlight scoring totals (117 points)
- Highlight organization achievements (championships, bowl games)
- Highlight "International Pathway Player"
